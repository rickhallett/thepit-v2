# Terminal and Safari

## Instructions
- As you work, keep track of your notes in TextEdit
- If you work longer than 5 minutes (based on your start time), wrap up and send what you have.

## Tasks
- [ ] Open Terminal and use drive to create a tmux session called workspace.
- [ ] Run 'ls -la ~/Desktop' and capture the output.
- [ ] Open Safari and take a screenshot of whatever page is loaded.
- [ ] Report both the directory listing and what you saw in Safari.

## Deliverables
- A report containing the Desktop directory listing from the tmux session and a description of the Safari page screenshot.
