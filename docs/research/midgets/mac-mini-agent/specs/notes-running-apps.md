# Notes running apps

## Instructions
- As you work, keep track of your notes in TextEdit
- If you work longer than 5 minutes (based on your start time), wrap up and send what you have.

## Tasks
- [ ] Open Notes app and create a new note.
- [ ] Use steer apps to find all currently running apps on this Mac.
- [ ] Type a summary of all currently running apps into the note.
- [ ] Take a screenshot of the completed note and report what you wrote.

## Deliverables
- A note in the Notes app summarizing all currently running apps, with a screenshot of the completed note.
