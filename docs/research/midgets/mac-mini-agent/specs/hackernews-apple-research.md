# HackerNews Apple research

## Instructions
- As you work, keep track of your notes in TextEdit
- If you work longer than 5 minutes (based on your start time), wrap up and send what you have.

## Tasks
- [ ] First note the current date and time in TextEdit.
- [ ] Open Safari and navigate to news.ycombinator.com.
- [ ] Use steer OCR to read and report the top 5 most popular posts.
- [ ] Scan the front page for any Apple device related news.
- [ ] If Apple news is found, click into the article, read the details, and report:
  - [ ] New spec offerings
  - [ ] Key features and focus of Apple's release
  - [ ] Target audience
- [ ] Write your findings in markdown format in TextEdit.

## Deliverables
- A markdown formatted document in TextEdit with the top 5 HackerNews headlines and any Apple device news details including specs, features, and release focus.
